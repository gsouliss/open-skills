# open-skills

- Usage:

    - Place skills in the following path (may be nested within a parent folder): 

    - The following path is for Windows (same for Linux, but different syntax and path structure).

```sh
C:\Users\user\.config\opencode\skills\SKILL.md
```

<br>

- Loading Skill:

    - After running opencode we execute the following in the respective folder:

```sh 
opencode .
/skills # Select skill
/dev-guidence # Press enter
```
